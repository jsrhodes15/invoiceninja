# Invoice Ninja clone
## Getting up and running with Docker
clone the repo and move to the Docker folder
```bash
cd Docker
```
ensure Docker is running on your machine, and fire up the services
```bash
docker-compose up
```
> this uses the images from dockerhub, local development will be configured and explain ASAP
